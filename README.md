# Produits / Catégories / Panier

Interface d'une boutique simple (comportant une page catalogue et une page panier).

## Installation et lancement

- Installer les dépendances:
  - npm install

- Lancer le projet:
  - npm run start
  
## Fait avec
- Redux
- React Query
- React Router
